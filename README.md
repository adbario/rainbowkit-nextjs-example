# RainbowKit Next.js Example App

A simple example app to get started with [RainbowKit](https://www.rainbowkit.com) using [Next.js](https://nextjs.org), [Alchemy](https://www.alchemy.com) and Goerli Ethereum Testnet.

## Getting Started

### Clone Repository

```shell
git clone git@github.com:adbario/rainbowkit-nextjs-example.git
```

### Install Dependencies

```shell
npm install
```

### Get Alchemy API Key

1. Sign up for an [Alchemy](https://www.alchemy.com) account.
2. Create a new app in Alchemy dashboard:
   - **CHAIN:** Ethereum
   - **NETWORK:** Goerli
3. Copy Alchemy API key to `.env.local` file which has been created automatically.

### Start Development Server

```shell
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the example app.
